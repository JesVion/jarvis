\# JARVIS AI Assistant 🤖

An Iron Man-style AI assistant built with HTML, CSS, and JavaScript.

## Features
- Voice recognition (listen to your commands)
- Text-to-speech responses
- Iron Man Arc Reactor visual design
- AI-powered responses using Groq API
- Local fallback commands

## How to Use

1. **Get a Groq API Key** (free, unlimited):
   - Go to https://console.groq.com
   - Sign up/Login
   - Click "Create API Key"
   - Copy your API key

2. **Add Your API Key**:
   - Open `src/main.js`
   - Find the line: `const GROQ_API_KEY = 'gsk_YourGroqAPIKeyHere';`
   - Replace with your actual API key

3. **Run the Project**:
   - Open `index.html` in a web browser
   - Click "Activate" to speak commands
   - Or click "Test Voice" to hear JARVIS speak

## Voice Commands
- "Hello" - JARVIS greets you
- "What time is it" - Tells you the time
- "Open browser" - Opens Google
- "Search for [topic]" - Searches Google
- "Who are you" - JARVIS introduces himself
- "Help" - Shows available commands

## Notes
- Use Chrome or Edge for best voice recognition
- Allow microphone access when prompted
- Internet connection required for AI responses
